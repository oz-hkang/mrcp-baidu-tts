<p>MPF - Media Processing Framework<br/></p><p><br/></p><p>The MPF library provides asynchronous interface for media processing with requests sent to the media processing engine, and responses or events sent back from the engine. The library is based on the following entities: media context, termination, audio/video stream, descriptor.<br/></p>
